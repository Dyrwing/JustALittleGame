# Just a little game

A simple memory game where you need to find matching pairs. Test your memory by flipping cards and matching them.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository

```bash
git clone <repository-url>
cd JustALittleGame
```

2. Install dependencies

```bash
cd react-app
npm install
```

### Running the Game

Start the development server:

```bash
npm run dev
```

The game will be available at `http://localhost:5173`
